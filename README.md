# СтройКонтроль

СтройКонтроль — это учебное веб-приложение для управления строительными проектами.  
Система позволяет работать с заказами, материалами, услугами и отчетами.

## Возможности
- авторизация пользователей;
- разграничение прав доступа по ролям;
- создание и просмотр заказов;
- работа со складом материалов;
- учет услуг;
- формирование отчетов;
- экспорт данных.

## Используемые технологии
- Python
- Flask
- Flask-Login
- HTML
- CSS
